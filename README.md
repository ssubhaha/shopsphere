# ShopSphere

A modern E-Commerce Product Catalog built using HTML, CSS, and JavaScript.

## Features

- Modular Frontend Architecture
- Client-Side Routing
- Responsive Design
- Product Catalog
- Product Search
- Shopping Cart
- Dark Mode
- Toast Notifications
- Performance Optimization
- GitHub Deployment
- Vercel Deployment
